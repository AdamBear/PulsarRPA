OS Environment
=================

Linux
-----

Update the system:

```bash
root@galaxyeye$ apt update
root@galaxyeye$ apt upgrade
```

Install necessary software:

```bash
root@galaxyeye$ apt install wget vim default-jdk maven git
```

Install mongodb:

Follow the official document: [install-mongodb-on-ubuntu](https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-ubuntu/)
