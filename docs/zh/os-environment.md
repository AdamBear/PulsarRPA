系统环境
=================

Linux[get-started](get-started)
-----

更新系统:

```bash
sudo apt update
sudo apt upgrade
```

安装必需软件:

```bash
sudo apt install wget vim default-jdk maven git
```

安装 MongoDB:

遵循如下教程: [install-mongodb-on-ubuntu](https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-ubuntu/)
